# NicoVisualEffects 3

NicoVisualEffects 3 は、オープンソースのモーショングラフィックスソフトです。

動画や画像、図形、テキスト等の複数のレイヤーを重ね、エフェクトやキーフレームによるアニメーションをつけることで、動きのある動画を作成することが出来ます。

## 動作環境

### 最低動作環境

* Intel AVX に対応した x64 CPU
* Windows 10 22H2、または Windows 11 22H2 以降
* .NET 10 がインストール済み、またはインストール可能である

### 推奨動作環境

* RAM 32GB 以上
* Intel AVX2、Intel AVX512 に対応した x64 CPU
* DirectX 12、シェーダーモデル 6.0 以上対応の GPU

## インストール方法

### (未インストールの場合) .NET 10 のインストール

.NET 10 をまだインストールしていない場合、先にインストールします。

https://dotnet.microsoft.com/ja-jp/download/dotnet/10.0

上記の URL から、.NET デスクトップ ランタイム、または SDK の Windows x64 用のものをダウンロードしてインストールします。

バージョンは最新のもの(バージョン番号の最後の数字が一番大きいもの)をダウンロードしてください。

### インストール

[Releases](https://github.com/mes51/NiVE3/releases) から NiVE3 の zip ファイルをダウンロード、任意のディレクトリに解凍します。

> [!TIP]
> zip ファイルを解凍する前に
>
> zip ファイルのプロパティを開いた際、「このファイルは他のコンピューターから取得した物です。...」というセキュリティのメッセージが出ていることがあります。
>
> このメッセージが出ている状態のまま解凍ファイルをした場合、NiVE3 が正常に動作しない事があります。
>
> zip ファイルを解凍する前に、ファイルのプロパティを確認し、該当のメッセージが出ている場合は、解凍前に右の「許可する」にチェックを入れ、OKを押してください。

## アンインストール方法

NiVE3 のファイルを削除します。

NiVE3 のインストール時、ディレクトリを作成していた場合は、ディレクトリごと削除するだけでアンインストール可能です。
